# Repository Profile: learning-embedded-iot

Last verified: 2026-06-20

## GitHub Description

Microcontroller and embedded IoT learning repo for MicroPython, Pico, ESP32, Zephyr, sensors, and MQTT.

## Topics

- `learning`
- `embedded`
- `iot`
- `micropython`
- `esp32`
- `zephyr`

## Audience

Engineers who want to learn firmware and IoT basics without losing the software-engineering habits: clear setup, reproducible examples, safe credentials, and documented hardware assumptions.

## First Milestones

1. Add MicroPython blink for RP2/Pico.
2. Add MicroPython sensor read with generated-data fallback.
3. Add Pico C SDK blink.
4. Add ESP-IDF Wi-Fi + MQTT example.
5. Add Zephyr blinky and native simulation notes.
6. Connect generated IoT events to `learning-data-stores`.

## Public Safety Checklist

- No Wi-Fi passwords.
- No cloud IoT tokens.
- No private device IDs.
- No unsafe high-voltage examples.
- Firmware outputs are ignored by Git.
